# Mix-ChatTTS

这是一个基于 ChatTTS 项目的扩展，添加了长文本处理功能。

## 项目简介

Mix-ChatTTS 是一个文本到语音转换工具，支持长文本的处理和转换。该项目基于开源的 ChatTTS 项目，提供了更强大的功能和更好的用户体验。
